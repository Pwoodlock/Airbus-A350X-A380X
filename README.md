## Getting Started
---
- X-Plane 11 / 12 
- KiCad 8 + (I'm on Version 9)
- FreeCad 1.0.0 upwards
- Visual Studio Code with STM32-CMAKE (Or your own IDE Preference) This might change with the new RP2350B 
- PFD Viewer / [Gerber Viewer](https://jlcpcb.com/RGE)
